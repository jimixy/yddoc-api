影刀RPA平台各模块API文档导航

核心模块

P
Package 模块
用于获取各类元素（网页元素、软件元素及图像元素）及对资源文件进行读写等操作。
  • 元素选择器 - 根据元素名称返回元素选择器对象
  • 图像选择器 - 根据图像库中图像元素的名称返回图像选择器对象
  • 资源文件对象 - 对资源文件进行读写操作
  • 支持文本、二进制文件操作
  • 剪切板文件操作
查看详细文档
A
ADO 模块
ADO 接口主要是针对数据的操作，如数据库的连接/断开、SQL 语句的执行。
  • 数据库连接管理
  • SQL 语句执行
  • 连接字符串配置
  • 超时控制
  • 结果集处理
查看详细文档
L
Logging 模块
用来记录程序运行中的日志信息，如：错误信息，调试信息，警告信息，普通信息。
  • 多级别日志记录
  • 调试信息输出
  • 错误信息记录
  • 警告信息提示
  • 普通信息记录

应用模块

D
DataBook 模块
对影刀内置的数据表格进行操作（如获取行内容、数据表格内写入行内容等）。
  • 行操作 - 获取、设置、追加、插入、删除行
  • 单元格操作 - 获取、设置单元格内容
  • 列操作 - 设置整列内容
  • 区域操作 - 获取、设置区域内容
  • 表格管理 - 清空、获取行数等
查看详细文档
D
Dialog 模块
调取各类对话框，实现与影刀机器人的交互。如消息对话框、自定义对话框。
  • 消息对话框 - 显示提示信息
  • 确认对话框 - 用户确认操作
  • 自定义对话框 - 多种控件支持
  • 数据表格对话框
  • 消息通知框
查看详细文档

Excel模块

E
Excel 模块
对Excel对象进行操作，如创建Excel对象、打开Excel文件等。
  • 创建Excel对象
  • 打开Excel文件
  • 获取当前激活的Excel
  • 支持Office、WPS、Openpyxl
  • 自动检测最佳方式
查看详细文档
W
WorkBook 模块
对工作簿进行各种处理，如获取Sheet页、激活Sheet页、创建Sheet页、保存为Excel文件、另存为Excel文件、执行宏、关闭工作簿等。
  • Sheet页操作 - 获取、激活、创建、删除、重命名
  • Sheet页复制 - 复制到当前或其他工作簿
  • 文件操作 - 保存、另存为、关闭
  • 宏操作 - 执行宏
  • 数据透视表操作
  • PDF导出功能
查看详细文档
S
WorkSheet 模块
对工作表对象进行处理，如获取Excel内容、写入Excel内容、清除Excel内容等。
  • 数据读取 - 单元格、行、列、区域
  • 数据写入 - 单元格、行、列、区域
  • 行操作 - 追加、插入、移除
  • 列操作 - 移除、插入
  • 内容管理 - 清空、区域清空
  • 区域操作 - 选择、复制、粘贴
  • 位置查找 - 可用行列查找
  • 显示控制 - 行列隐藏
查看详细文档

网页模块

W
Web 模块
对 web 网页进行各式操作，如打开网页、关闭网页。
  • 创建网页 - 支持多种浏览器
  • 获取网页 - 按标题、URL获取
  • 获取激活网页
  • 获取所有网页
  • Cookie管理
  • 关闭所有网页
  • 文件上传下载对话框处理
查看详细文档
B
Browser 模块
对网页元素、网页进行的各种处理，如网页激活、获取网页标题、网页重新加载、取消网页加载等。
  • 网页信息获取 - URL、标题、文本、HTML
  • 网页导航 - 激活、跳转、前进、后退
  • 加载控制 - 重新加载、停止加载、等待完成
  • JavaScript操作 - 执行JS脚本
  • 网页滚动 - 滚动到指定位置
  • 对话框处理 - 处理网页对话框
  • 网络监听 - 监听网页请求
  • 元素查找 - 多种选择器支持
  • 表格数据提取
查看详细文档
E
Element 模块
对网页元素进行处理，如获取相似网页元素、查找目标元素父/子元素、点击元素等。
  • 元素关系 - 父元素、子元素、兄弟元素
  • 鼠标操作 - 点击、双击、悬停、拖拽
  • 输入操作 - 填写输入框、剪切板输入
  • 内容获取 - 文本、HTML、值
  • 复选框操作 - 选中、取消选中
  • 下拉框操作 - 单选、多选、按索引选择
  • 属性操作 - 设置、获取属性
  • 状态判断 - 可用性、可见性
  • 截图操作 - 元素截图
  • 文件操作 - 上传、下载
  • 表格操作 - 提取表格数据
